# JAVA JDK 설치 방법입니다.

## 1. JAVA 11버전 설치 방법 (Ubuntu)
(1) OS 패키지 정보 업데이트
* 명령어
```
$ sudo apt update
```

(2) JAVA 11버전(OpenJDK) 설치
* 명령어
```
$ sudo apt install -y default-jdk
```

(3) JAVA 버전 확인
* 명령어
```
$ java -version
```
