# CLI (Commend Line Interface)

> 유닉스 shell (sh,zsh,bash 등) , 윈도우 cmd

- bash / shell 기본 명령
  - pwd : 지금 위치 (폴더 위치)
  - ls : 현위치 파일
  - clear : 창 깨끗하게~
  - mkdir : 폴더 만들기
  - touch : 파일 만들기
  - rm : 파일 지우기
  - echo : 출력하기



- git 명령어

  - git init
  - git add
  - git commit -m "메시지"
  - git config --global user.email "   "
  - git config --global user.name "   "
  - git log
