### linux tar,gz 압축 및 해제
#### tar로 압축하기
- tar -cvf [파일명.tar] [폴더명]

### tar 압축 풀기
- tar -xvf [파일명.tar]

### tar.gz로 압축하기
- tar -zcvf [파일명.tar.gz] [폴더명]

### tar.gz 압축 풀기
- tar -zxvf [파일명.tar.gz]



옵션 설명
-c 파일을 tar로 묶음
-p 파일 권한을 저장
-v 묶거나파일을 풀 때 과정을 화면으로 출력
-f 파일 이름을 지정
-C 경로를 지정
-x tar 압축을 풂
-z gzip으로 압축하거나 해제함
