# 검색및정렬_이진 검색이 삼항 검색보다 선호되는 이유

- 이진검색(Binary Search)
  - T(n) = T(n/2) + 2, T(1) = 1
- 삼항검색(Ternary Search)
  - T(n) = T(n/3) + 4, T(1) = 1

###### 시간 복잡도

이진 탐색을위한 시간 복잡도 = 2clog 2 n + O (1) 

삼항 탐색을위한 시간 복잡도 = 4clog 3 n + O (1)

**최악의 경우에 이진 검색보다 더 많은 비교를 수행하기 때문에 2진 검색이 더 낫다**
