## 알고리즘

- 자세해야 한다.
- "**매우**" 자세해야 한다.
- 가정을 확인 할때는 **본명제와 대우를 확인하면 된다.**



※ 생전 처음보는 문제를 보았을때!!!

→ 어떻게 풀지 그냥 생각해본다.

→ 잘모르겠으면 조건을 지키면서 하나씩 대입, 빠짐없이 실행 해보면서 규칙을 찾거나 해결방법을 고민해보자.

(문제풀이) > 조건을 세분화 > 더 더 세분화
