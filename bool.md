# Built-in Functions_bool()

- 사용법 :

  ` bool(x)`

  참, 거짓을 확인하고 싶은 인자를 넣는다.

- 기능 :

  - 인자가 가지는 리턴값을 확인할 수 있다.
  - 인자가 들어 있다면 True / 없다면 False
  - 기본적으로 기본적인 Truth testing procedure 를 따라 변형된다.

- 리턴값 :

  부울형 (True / False)

```python
bool(1)			#   True
bool(0)			#   False
a=[]			
bool(a)			#   False
b=[1]
bool(b)			#   True
```
